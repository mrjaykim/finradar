# ADR-006: 리스크 예측 모델 아키텍처 - 예측 트랙과 참고 스코어 트랙 분리

## 상태
결정됨 (2026-07-09)

## 배경
상장폐지 라벨([ADR-005](ADR-005-delisting-as-risk-label.md))로 이진분류 모델을 만들 때,
재무비율(부채비율, 이자보상배율 등)을 라벨 정의에도 쓰고 예측 피처에도 쓰면 순환논리에 빠질
위험이 있음을 발견했다. 즉 모델이 "이 기업이 위험한가"를 학습하는 게 아니라 "우리가 정의한
재무비율 규칙을 재현할 수 있는가"만 학습하게 되어, 예측력이 규칙 자체를 넘어서지 못한다.

## 검토한 대안

1. 재무비율 기반 규칙(이자보상배율 등)을 라벨로 삼고 같은 재무비율을 피처로 사용
2. 상장폐지만을 유일한 라벨/출력으로 사용하는 순수 이진분류
3. 트랙 분리: 트랙 A(상장폐지 라벨 기반 예측 모델, 재무비율은 피처로만 사용) + 트랙 B(Altman
   Z-Score, Piotroski F-Score 등 학술 공식 기반 참고 스코어를 별도 병행 산출)

## 선택
3번 채택.

## 선택 이유
- 라벨과 피처의 순환논리를 원천 차단
- 트랙 B는 라벨이 필요 없는 고정 수식이라 데이터 부족 문제와 무관하게 항상 계산 가능
- 최종 대시보드에서 "ML 예측 리스크 72점 vs Altman Z-Score 68점"처럼 병행 제시 가능해
  설명력이 높아짐

## 트레이드오프
두 트랙을 유지보수해야 하는 복잡도가 늘어난다. Altman Z-Score 등 학술 공식은 원래 미국
상장사 기준으로 만들어져 한국 시장에 그대로 적용 시 계수 보정이 필요할 수 있다.
